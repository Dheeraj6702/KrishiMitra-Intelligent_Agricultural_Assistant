# KrishiMitra-Intelligent_Agricultural_Assistant
KrishiMitra is an AI-powered agriculture assistant for Indian farmers. It has 4 modules — Crop Recommendation (inputs: N, P, K, Temperature, Humidity, Rainfall, Soil Type), Fertilizer Suggestion (inputs: N, P, K, pH, Organic Carbon, Crop Type), Plant Disease Detection (image input, severity, treatment), and Government Schemes filter.
